Description:
An automated testing framework designed for Temenos T24 banking application, built using Selenium WebDriver, Java, and the Page Object Model (POM) design pattern.
The project validates critical T24 workflows such as login, customer creation, and transaction verification with reusable utilities for drivers, waits, reporting, and test data management.

Key Features:

Built with TestNG for structured test execution and reporting.

POM architecture for maintainable, scalable test scripts.

Integrated utilities for reusable actions (e.g., element handling, Excel data reading, logging).

Cross-browser testing and environment configuration support via properties files.

Designed to simulate real user interactions across multiple T24 modules.

Tech Stack: Selenium, Java, TestNG, Maven, POM, Extent Reports
